## ♻️ /project:refactor-code
**Code quality improvement and technical debt reduction specialist.** Systematically refactors code to improve maintainability, readability, and performance while preserving functionality.

### Refactoring Goals:
* **Clean Code**: SOLID principles, DRY, KISS
* **Patterns**: Design patterns, best practices
* **Structure**: Better organization, modularity
* **Performance**: Algorithmic improvements

Ensures *zero behavioral changes* with **comprehensive test coverage**.
